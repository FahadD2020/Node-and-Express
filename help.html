Method Path Example Description
GET /Users /Users GET all users in response.
POST /Users Not applicable Submit button on /Users does a POST. Return the created entity or 400 on bad request.
GET /Users/{userId} /Users/12 Find and return the user with given Id. 404 if not found
PATCH /Users/{userId} /Users/12 Allow to update data (except ID) of an existing user. Returns 404 if user not found.
DELETE /Users/{userId} /Users/12 Delete the user with given ID. Returns 404 if user not found.
